# 学生信息管理系统

## 快速开始

### 环境要求

- GCC 7.0+ 或 Clang 5.0+
- GNU Make

### 编译项目

```bash
# 进入项目目录
cd Student_manage_system

# 编译项目
make

# 或者编译并运行
make run
```

### 运行程序

```bash
# 直接运行可执行文件
./student_management_system
```

## 使用说明

启动程序后，使用数字选择菜单功能：

1. **添加学生** - 输入学生基本信息
2. **删除学生** - 根据学号删除学生
3. **修改学生信息** - 更新学生信息
4. **查询学生信息** - 按学号查询
5. **显示所有学生** - 列出所有学生
6. **保存到文件** - 导出为CSV格式
7. **设置学生成绩** - 设置各科成绩
8. **查询学生成绩** - 查看成绩和平均分
9. **退出系统**

## 常用命令

```bash
make        # 编译项目
make run    # 编译并运行
make clean  # 清理构建文件
make debug  # 编译调试版本
```

## 数据文件

- 学生数据保存在 `students.csv` 文件中
- 构建文件在 `build/` 目录中

## 功能介绍

分成三层：储存结构，算法层面，交互层面
    1. 储存结构：使用链表存储学生信息（stl），由于采用stl，不再单独封装。
    2. 算法层面：实现基本的增删改查操作
    3. 交互层面：提供命令行界面，用户可以通过输入命令来操作系统
